# Next.js with Typescript + Tailwind CSS

## To Run Locally

```bash
git clone https://github.com/default-sameer/nextjs-typescript-tailwind.git                                                                                                                    
```

### Install Dependecies

```bash
$ yarn install 
or
$ npm install
```

### Run the App

```bash
$ yarn dev
or 
$ npm run dev
```
